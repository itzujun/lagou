# lagou 爬取招聘职位

### 运行
```
python install lagouanay.py  # 网络爬虫数据
```

```
python install lagouanay.py  # pandas分析数据并可视化
```

#### 如下错误提示

```
{'status': False, 'msg': '您操作太频繁,请稍后再访问', 'clientIp': '106.2.234.97', 'state': 2402}
```
#### 原因

```
Cookie，Referer，User-Agent 需爬虫手动设置，后台有监测返爬虫机制
```
